Cezerin

Cezerin is an open source API-based solution for e-commerce.
Shipped with REST API, web dashboard and web store.

View on GitHub Download .zip Download .tar.gz

Why we build Cezerin?

As a geeks we see how quickly world is changing. Consumer can be any kind of device or service. This can be drones, bot or personal assistant. We know how modern technologies can help in development and deliver new opportunities to customers. That's why we build Cezerin and we want to share it with world.

Full-Stack JavaScript

API (JSON REST) - NodeJS, MongoDB
Dashboard (client side) - ReactJS, Redux
Store (server rendering) - NodeJS
